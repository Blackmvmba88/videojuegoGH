# To-Do List Application

A simple and efficient To-Do List app built with React, featuring persistent storage via the browser's localStorage.

## Features

- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Your tasks are saved in your browser and persist after refresh

## How to Run

1. Clone the repo
2. Run `npm install`
3. Start with `npm start`

## Tech Stack

- React
- TypeScript
- LocalStorage API
